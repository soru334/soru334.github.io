ペース貯金 v5.10

修正内容
- 時給モードの年累計計算を修正
  - 「月の出勤日数から推定」でも年間/累計の給与計算が崩れないよう修正
- 休み日の柔軟設定を追加
  - 祝日は休みにする
  - 個別の日付ごとに「休み」「出勤日」を上書き
- 基本設定を折りたたみ対応に変更
- サマリーを一本化
  - 累計
  - 前年
  - 今年
  - 来年
- カレンダーを上に移動して見やすく調整

補足
- 上の大きい金額は今の給料日区間の給与
- サマリーは選んだ期間の給与/ボーナス/支出/収支
- 現在の残高はこれまで通り
  初期銀行残高 + 給与の積み上がり + 臨時収入 - 支出 - 固定費
