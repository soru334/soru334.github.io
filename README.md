[README_pace_saver_v5_6.txt](https://github.com/user-attachments/files/26260132/README_pace_saver_v5_6.txt)ペース貯金 v5.6

追加したもの
- 時給入力時の逆算方法を追加
  - 勤務曜日から推定
  - 月の出勤日数から推定（不規則シフト向け）
- 月の出勤日数を直接入力できる欄を追加
- プレビューにも月の出勤日数を表示
- 年間給与・残高計算にも新しい逆算方法を反映

補足
- 月の出勤日数を使う場合は、時給・勤務時間・休憩・出勤日数から推定月給を計算します
- このモードでは、給料日から給料日までの区間全体に対して金額がなめらかに積み上がる表示になります
- カレンダーの日付タップ詳細はそのまま利用できます
